## 🌟 What is AptIQ?
**AptIQ** is a full-stack MERN application that makes technical interviews and aptitude testing easy. It combines standard multiple-choice questions with a **live, in-browser Python coding environment**. Students can write, run, and submit code seamlessly, while faculty can easily track their performance.

## 🚀 Key Features
* 👨‍💻 **In-Browser IDE:** Write and run Python code safely in the browser using WebAssembly (Pyodide).
* 🔴 **Live Proctoring:** Automatic webcam monitoring during coding rounds to ensure fairness.
* 🔐 **High Security:** Secure OTP email verification (via Brevo) and JWT-based logins.
* 📊 **Smart Grading:** Auto-evaluation of student code against hidden test cases.
* 🎓 **Role-Based Portals:** Dedicated dashboards tailored for Students and Faculty.

## 📁 Project Structure
* **`/frontend`** - React.js UI, Monaco Editor, and Pyodide sandbox.
* **`/backend`** - Node.js/Express.js API, MongoDB schemas, and Authentication.
